Web Security

How would you advise a potential client to secure his/her website from web security attacks? Note: the word count for your answer should not be greater than 250 words

There are several methods of both attacking and securing a website from attacks.The most common threats faced is XSS, SQL Injection, and Cross-Site Request Forgery. Most of these threats have different ways of fighting them, but the most standard practices include: Data sanitization, Secure authorization and authentication, user timeouts, CI/CD and network security. These practices in combination are proven to be very efficient in warding off web security threats.
